About Me

My love for coding started over 10 years ago back when I was in high school. I would teach myself HTML and CSS by reading W3Schools and source code on other websites then testing it out on my own site. I would code everything using Microsoft FrontPage and design layouts in Adobe Photoshop.

At the time, JavaScript was seen as the programming language for amateurs. You could make silly effect with it like the cursor could be changed into a sprite. But now JavaScript is king of the internet and the ecosystem for the language has exploded. There are new frameworks and libraries being published everyday. There are so many concepts that have been adopted since then.

That is why I decided to go to the Georgia Tech Coding Bootcamp. It is an intensive 6 month bootcamp where where I am learning JavaScript, and frameworks like Node, Express, and React. I am also learning concepts like programming best practices, MVC, and TDD.
